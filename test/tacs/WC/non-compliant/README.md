# TACs WC (SIGMET Tropical Cyclone) Non-Conformes

Ce dossier contient des fichiers SIGMET cyclones tropicaux qui ne sont pas conformes aux standards OACI.

## Problèmes courants

### Format de distances non standard

- Formats comme `250NM`, `240KM`, `075NM` non reconnus par la grammaire actuelle

### Grammaire SIGMET TC incomplète

La grammaire SIGMET TC actuelle ne supporte pas encore tous les formats de messages.
Ces fichiers seront réintégrés une fois la grammaire complétée.
