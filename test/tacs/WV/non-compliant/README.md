# TACs WV (SIGMET Volcanic Ash) Non-Conformes

Ce dossier contient des fichiers SIGMET cendres volcaniques qui ne sont pas conformes aux standards OACI.

## Problèmes courants

### Format de coordonnées et distances non standard

- Formats comme `W07913-` ou `-S0200` non reconnus
- Noms de volcans avec parenthèses non supportés

### Grammaire SIGMET VA incomplète

La grammaire SIGMET VA actuelle ne supporte pas encore tous les formats de messages.
Ces fichiers seront réintégrés une fois la grammaire complétée.
