# TACs WA (AIRMET) Non-Conformes

Ce dossier contient des fichiers AIRMET qui ne sont pas conformes aux standards OACI.

## Problèmes courants

### Formats de niveau de vol non standard

- Formats comme `000/0500FT`, `6000FT`, `5000FT` non reconnus

### Grammaire AIRMET incomplète

La grammaire AIRMET actuelle ne supporte pas encore tous les formats de messages.
Ces fichiers seront réintégrés une fois la grammaire complétée.

## Fichiers concernés

- WAAB31_LATI_*.txt - Messages AIRMET albanais
- WABZ23_SBGL_*.txt - Messages AIRMET brésiliens
- Et autres...
